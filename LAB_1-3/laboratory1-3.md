# Лабораторна робота 1-3

Ця лабораторна робота реалізує простий конструктор резюме.

## Що робить програма

- Відображає кнопку для створення нового резюме.
- Запитує у користувача дані через `prompt()`:
  - ім'я;
  - email;
  - вік;
  - компанію;
  - посаду;
  - роки досвіду;
  - навички (через кому).
- Створює розділи резюме на основі введених даних.
- Відображає сформоване резюме у блоці `#resumeOutput`.
- Зберігає дані в `localStorage`.

## Основні функції файлу `laboratory1-3.js`

- `createValidator(errorMessage)`
  - повертає карровану функцію-валідатор.
  - використовується для створення перевірок на непорожнє значення та позитивне число.

- `isNotEmpty` та `isPositiveNum`
  - `isNotEmpty` перевіряє, що текст не пустий і містить символи після обрізки.
  - `isPositiveNum` перевіряє, що значення є числом і більше за 0.

- Клас `ResumeSection`
  - базовий клас для секцій резюме.
  - зберігає заголовок секції та створює базовий DOM-елемент.

- Клас `PersonalInfo`
  - розширює `ResumeSection`.
  - містить `name`, `email` та приватне поле `#age`.
  - перевіряє всі поля і рендерить блок з особистою інформацією.

- Клас `Experience`
  - розширює `ResumeSection`.
  - зберігає компанію, роль та роки досвіду.
  - рендерить опис досвіду роботи.

- Клас `Skills`
  - розширює `ResumeSection`.
  - приймає рядок навичок, розбиває його на масив за комами та очищує пробіли.
  - рендерить навички у вигляді списку з роздільниками `•`.

- Клас `Resume`
  - зберігає масив секцій.
  - додає секції та рендерить їх у контейнер за `containerId`.
  - зберігає результат у `localStorage` під ключем `myResume`.

## Структура HTML

- `button#createResumeBtn` — кнопка запуску процесу створення резюме.
- `div#resumeOutput` — контейнер для відображення згенерованого резюме.
- `link rel="stylesheet" href="laboratory1-3.css"` — підключає стилі оформлення.
- `script src="laboratory1-3.js" defer` — підключає логіку.

## Додатково

Сторінка використовує CSS з файлу `laboratory1-3.css` для оформлення контейнера, кнопки та секцій резюме. При помилці валідації користувача показується повідомлення через `alert()`.
