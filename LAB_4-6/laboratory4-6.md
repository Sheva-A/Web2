# Лабораторна робота 4-6

Ця лабораторна робота реалізує просту веб-додаток для перегляду бази телевізійних шоу з API `TVMaze`.

## Що робить програма

- Завантажує список шоу з API за адресою `https://api.tvmaze.com/shows`.
- Відображає кожне шоу у вигляді картки з постером, назвою, рейтингом та жанрами.
- Дозволяє шукати шоу за назвою через текстове поле пошуку.
- Дозволяє сортувати результати за алфавітом або за рейтингом.
- Відображає повідомлення про помилку у разі проблем із завантаженням даних.

## Основні функції файлу `laboratory4-6.js`

- `fetchMovies()`
  - Завантажує дані з API.
  - Перевіряє статус відповіді та обробляє помилки.
  - Зберігає отриманий масив в `allMovies` і викликає `displayMovies()`.

- `displayMovies(movies)`
  - Очищує контейнер `moviesContainer`.
  - Формує HTML-картки для кожного шоу.
  - Вставляє картки у DOM.

- `handleProcessData()`
  - Фільтрує список `allMovies` за введеним запитом у полі `searchInput`.
  - Сортує фільтрований список за вибором у `sortSelect`:
    - `alphabetical` — за назвою (А-Я);
    - `rating` — за рейтингом від високого до низького.
  - Викликає `displayMovies()` для оновлення відображення.

- `showError(message)`
  - Відображає текстову помилку у блоці `errorMessage`.
  - Змінює клас для показу повідомлення.

## Структура HTML

- Поле для пошуку `#searchInput`.
- Вибір сортування `#sortSelect`.
- Контейнер `#moviesContainer` для карток шоу.
- Блок `#errorMessage` для помилок.

## Додатково

Сторінка також використовує CSS з файлу `laboratory4-6.css` для оформлення карток, розмітки і повідомлень про помилки.
